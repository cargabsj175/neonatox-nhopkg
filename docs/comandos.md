# 4\. Main commands

Command| Description  
---|---  
`-i, --install`| Install a local binary package  
`-S, --super-install`| Install a package from a remote repository  
`-b, --build`| Build and install a local source package  
`-y, --upgrade`| Upgrade all installed packages  
`-r, --remove`| Remove an installed package  
`-l, --list`| List installed packages  
  
## Examples
    
    
    sudo nhopkg -S gimp
    sudo nhopkg -y
    nhopkg -n gimp
