4. Main commands

CommandDescription
-i, --installInstall a local binary package
-S, --super-installInstall a package from a remote repository
-b, --buildBuild and install a local source package
-y, --upgradeUpgrade all installed packages
-r, --removeRemove an installed package
-l, --listList installed packages

Examples

sudo nhopkg -S gimp
sudo nhopkg -y
nhopkg -n gimp
